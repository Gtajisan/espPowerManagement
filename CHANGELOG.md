## [2.1.1] - 2025-12-15

### Added
- Combined power management with WiFi security testing
- WPS vulnerability detection with 576+ device signatures
- Manufacturer identification using OUI database
- WiFi standard detection (WiFi 4/5/6)
- Modern dark-themed Bootstrap 5 dashboard
- REST API endpoints (`/api/scan`, `/api/stats`)
- Bootstrap Icons integration
- Comprehensive documentation suite
- GitHub Actions CI/CD workflow
- Multiple board support (D1 Mini, NodeMCU)

### Changed
- Complete UI redesign with dark theme
- Improved navigation with icons
- Enhanced statistics display
- Better mobile responsiveness
- Updated to Bootstrap 5.3.2

### Security
- Added security disclaimer
- Educational use warnings
- Ethical usage guidelines

## [2.0.1] - 2025-11-02 (espPowerManagement base)

### Added
- Power monitoring system
- Automatic ON/OFF detection
- Duration calculation
- Statistics dashboard (Today, 7-day, 15-day, Monthly)
- Bootstrap 5 responsive interface
- WiFi Station + AP fallback mode
- SPIFFS persistent storage
- Monthly auto-reset

### Fixed
- Bootstrap 5 navbar toggle
- Mobile responsive issues
